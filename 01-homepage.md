# Supermemory Homepage — Wireframe + Copy

> The homepage is a **company hub**. It educates new visitors on what Supermemory is,
> makes the audience split crystal clear, and routes people to the right product page.
> Light mode. Content-first. No jargon. Short page.

---

## NAV BAR

```
[⊹ supermemory]   Enterprise   Plugins   Nova   Docs   Blog   Pricing   [𝕏 20K]   [Get Started →]
```

- "Enterprise", "Plugins", "Nova" are the 3 products — dead simple
- "Get Started" replaces "Try for free" (lower friction language)
- Remove: "Research", "Use Cases", "About" (noise for a hub page)


**Headline:** `Your AI isn't intelligent until it remembers`
- This was the strongest line on the current site. Julian loved it. Lead with it.

**Subhead:** `Supermemory is the memory layer for AI — from personal plugins to enterprise infrastructure. One company, every layer of memory.`

**CTAs:** Two equal buttons. No hierarchy — let the visitor self-select.
- `Explore Enterprise` → /enterprise
- `Explore Plugins` → /plugins

**Why this works:**
- No jargon. A Twitter visitor landing here instantly gets it.
- The dual CTA creates the audience split in the first 3 seconds.
- "Memory layer for AI" is the one concept they need to understand.

## PAGE SUMMARY

| Section | Purpose | Height target |
|---------|---------|---------------|
| Hero | Educate + route | ~40vh |
| The Split | Audience differentiation | ~50vh |
| Trust Bar | Social proof (logos) | ~10vh |
| Twitter Testimonials | Viral proof | ~40vh |
| Nova Teaser | Awareness | ~15vh |
| Pricing Overview | Clarity on products | ~50vh |
| Footer | Navigation | standard |

**Total estimated:** ~4-5 scroll lengths. WAY shorter than current ~19,500px page.

**Key principles applied:**
1. Audience split happens in first 2 folds (problem #1 solved)
2. Hero is the strongest existing line, no jargon (problem #4 solved)
3. Cross-agent memory story is front and center in the Plugins card (problem #5 solved)
4. Pricing clearly maps to products (problem #8 solved)
5. Page is short and non-repetitive (problem #7 solved)
6. Nav reflects actual products, not confusing labels (problems #2, #3 solved)
