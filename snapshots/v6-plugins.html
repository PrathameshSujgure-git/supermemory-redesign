Built on MCP

Open protocol.
No lock-in.

What is MCP? Model Context Protocol is an open standard (created by Anthropic) that lets AI tools connect to external data sources. Think of it as USB-C for AI — one universal plug.

Supermemory is a standard MCP server. It works with any MCP-compatible client today and any that launches tomorrow. Your data is never locked in.

  • ✓ Standard MCP server — any client works
  • ✓ Self-hostable or use our cloud
  • ✓ REST API for custom integrations
  • ✓ Export your data anytime (JSON)

Privacy & Data Ownership

You own your data. Period.

We built Supermemory for developers who care about where their data lives. Here's exactly how it works.

Encrypted at rest and in transit

All data encrypted with AES-256 at rest and TLS 1.3 in transit. Cloud-stored memories are encrypted with your account key — we cannot read your memories.

Self-host everything

Run the entire stack on your own infrastructure with Docker. MIT licensed, fully open source. Zero data ever leaves your network.

Full data portability

Export all memories as JSON anytime. Delete your account and all data is permanently removed within 24 hours. No data lock-in, no hostage games.

What gets stored, and what doesn't

Supermemory only stores what you explicitly save. It never silently collects your code or conversations.

Stored (only when you save)
  • ✓ Preferences you tell it to remember
  • ✓ Code patterns you explicitly save
  • ✓ Web pages saved via Chrome extension
  • ✓ Debug solutions you ask it to keep
  • ✓ Project-level context and tags
Never stored
  • × Your source code or repository contents
  • × Full conversation history with AI tools
  • × File system access or directory structures
  • × Environment variables or secrets
  • × Usage analytics or telemetry
